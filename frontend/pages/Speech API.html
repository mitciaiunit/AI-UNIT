What is Artificial Intelligence?

Artificial Intelligence, also called AI, is a technology that allows computers and systems to perform tasks that normally require human intelligence. These tasks can include reading, writing, recognising patterns, making predictions and helping people make better decisions.

In Mauritius, AI can support public services, education, healthcare, transport, agriculture, tourism and business innovation. However, AI must be used responsibly so that it remains fair, safe, transparent and useful for citizens.

Why is a Read Aloud Button Useful?

A Read Aloud button helps users listen to website content instead of reading it on screen. This can support blind users, low vision users, elderly users, students and anyone who prefers listening to information.

This demo uses the browser's Speech Synthesis API. When the user clicks the Read Aloud button, the website collects the text from this article and converts it into spoken audio.

How it works

The JavaScript code takes the text from the page, creates a speech object using SpeechSynthesisUtterance, then asks the browser to speak it using window.speechSynthesis.speak().
